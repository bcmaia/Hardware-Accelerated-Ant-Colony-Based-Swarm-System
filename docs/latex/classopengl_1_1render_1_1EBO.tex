\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classopengl_1_1render_1_1EBO}{EBO}} class represents an Elements Buffer Object in Open\+GL. 

An Elements Buffer Object (\mbox{\hyperlink{classopengl_1_1render_1_1EBO}{EBO}}) is used to store index data that specifies the order in which vertices should be rendered. This class provides methods to bind, unbind, and erase the \mbox{\hyperlink{classopengl_1_1render_1_1EBO}{EBO}}.
