\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classopengl_1_1render_1_1VBO}{VBO}} class represents a Vertex Buffer Object in Open\+GL. 

A Vertex Buffer Object (\mbox{\hyperlink{classopengl_1_1render_1_1VBO}{VBO}}) is a buffer that stores vertex data on the GPU. This class provides methods to create, resize, update, bind, unbind, and erase a \mbox{\hyperlink{classopengl_1_1render_1_1VBO}{VBO}}.
